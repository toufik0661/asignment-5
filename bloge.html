Question-1: What are the different ways to select an element in DOM?

In DOM we can select an element by many ways like getElementById(), getElementsByClassName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By using innerHTML() method we can see the inner html of an element and by using innerText() method we can see the inner text of an element and by using textContent() method we can see the inner text of an element with hidden text

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশনে আমরা একটি প্যারেন্ট নোডে একটি ইভেন্ট লিসেনার বরাদ্দ করি যার ফলে আমরা এর চাইল্ড এলিমেন্ট দ্বারা ট্রিগার হওয়া ইভেন্টগুলি দক্ষতার সাথে পরিচালনা করতে পারি।

Question-4: What is event bubbling in the DOM?

যখন আমরা ব্রাউজারের যেকোনো জায়গায় ক্লিক করি তখন ইভেন্টটি তাৎক্ষণিকভাবে ক্লিক করা টার্গেটটি চিনতে পারে না। এটি ইভেন্ট প্রোপাগেশন নামক একটি ক্রম অনুসরণ করে। প্রথমে এটি ক্যাপচারিং ফেজের মধ্য দিয়ে যায়: window > document > html > head > body > h > ul > li। যখন এটি টার্গেট এলিমেন্টে (li) পৌঁছায়, তখন বাবলিং ফেজ শুরু হয়, যেখানে ইভেন্টটি ব্যাক আপ প্রচার করে: li > ul > body > html > document > window। যদি এই যেকোনো পর্যায়ে ইভেন্ট লিসেনারের যোগ করা হয় তাহলে সেগুলি সেই অনুযায়ী ট্রিগার হবে।
